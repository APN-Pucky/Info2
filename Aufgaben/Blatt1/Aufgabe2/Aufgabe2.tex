NAND:\\*
NOT A = A NAND A\\*
A AND B = NOT (A NAND B) = (A NAND B) NAND (A NAND B)\\

NOR:\\*
NOT A = A NOR A\\*
A AND B = NOT A NOR NOT B = (A NOR A) NOR (B NOR B)\\
